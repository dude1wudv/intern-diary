# 模板变量填充

上级：[[_M1-模板系统]]
下级：无
依赖：`UserContext.profile_values`

---

## 场景

报告导出时需要把姓名、班级、学号、日期范围、报告标题写入 Word。

## 触发

生成 docx 时触发。

## 逻辑

- 复用现有 `{{姓名}}`、`{{班级}}`、`{{学号}}`。
- 新增 `{{开始日期}}`、`{{结束日期}}`、`{{报告类型}}`、`{{标题}}`。
- 渲染后扫描残留 `{{...}}`，写入 validation。

## 状态 / 边界

- profile 为空则变量填空并产生 warning，不阻断导出。
