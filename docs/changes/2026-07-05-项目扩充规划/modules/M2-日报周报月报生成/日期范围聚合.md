# 日期范围聚合

上级：[[_M2-日报周报月报生成]]
下级：无
依赖：现有 `day_dir`、`collect_day_source`

---

## 场景

生成周报/月报/总结时，需要把多天素材合并成稳定上下文。

## 触发

调用 `POST /api/actions/generate-report`。

## 逻辑

- 校验 start/end 日期。
- 按日期升序读取每一天 raw_text、sorted_notes、image_descriptions、diary_draft。
- 每天用标题分隔，空天跳过或标注“无素材”。
- 限制最大范围：weekly 建议 14 天内，monthly 建议 45 天内，summary 可更长但要截断/摘要。

## 状态 / 边界

- 完全无素材：返回 400，提示先添加素材。
