# 周报月报生成

上级：[[_M2-日报周报月报生成]]
下级：无
依赖：[[日期范围聚合]]、[[模板注册表]]、[[报告校验]]

---

## 场景

用户需要提交一周或一个月的实习成果。

## 触发

Web/Android 调用 `POST /api/actions/generate-report`，type 为 `weekly` 或 `monthly`。

## 逻辑

- 聚合日期范围素材。
- 按 type 选择 prompt 和 word_count。
- 生成 Markdown。
- 渲染 Word。
- 写入 `reports/<report_id>/metadata.json`。

## 状态 / 边界

- LLM 失败：不覆盖已有成功报告。
- 内容太短：生成 warning，允许用户补充要求后重试。
